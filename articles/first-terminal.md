# 我的第一个终端项目

**日期**: 2025-01-05

这是我创建的第一个终端风格的网站项目。记录一下这个项目的设计和实现过程。

## 项目背景

一直很喜欢命令行界面的简洁和高效，于是萌生了创建一个仿 CLI 风格个人网站的想法。

## 设计理念

### 简洁至上

去除所有不必要的元素，只保留核心功能。

### 命令行交互

使用熟悉的命令行交互方式，提供独特的用户体验。

### 深色主题

采用深色终端风格，护眼且专业。

## 技术实现

### 左右分栏布局

左侧显示个人信息，右侧是 CLI 终端模拟器。

### 命令系统

实现了三个基础命令：

- `pwd` - 显示当前目录
- `whoami` - 显示用户信息
- `ls` - 列出文章列表

### Markdown 解析

使用 `marked.js` 库在前端解析 Markdown 格式的文章。

## 遇到的挑战

### 输入框自动聚焦

需要确保用户点击终端区域时输入框能够自动聚焦。

### 文章在新标签页打开

使用 `target="_blank"` 属性实现文章在新标签页中打开。

### Markdown 元数据提取

需要从 Markdown 文件中提取标题和日期信息。

## 未来计划

- 添加命令历史记录
- 实现 Tab 自动补全
- 添加更多命令（如 clear, help 等）
- 优化移动端体验

## 项目收获

通过这个项目，我学到了：

1. 更深入的 CSS Flexbox 布局知识
2. JavaScript 事件处理和 DOM 操作
3. Markdown 解析技术
4. 静态网站的构建和部署

## 总结

这是一个简单但有趣的项目。虽然功能不多，但它提供了一个独特的交互体验。
