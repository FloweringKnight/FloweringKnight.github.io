# 如何使用 CLI 工具

**日期**: 2025-01-15

欢迎来到 FAKE-CLI！这是一个仿命令行风格的个人网站。本文档将介绍如何使用这个网站。

## 简介

FAKE-CLI 是一个基于 Web 的命令行模拟器，它提供了一个独特的交互体验来浏览个人信息和文章内容。

## 可用命令

### pwd

显示当前工作目录。

```bash
user@fake-cli:~$ pwd
/user/
```

### whoami

显示当前用户信息。

```bash
user@fake-cli:~$ whoami
I don't know, sorry~
```

### ls

列出所有可用的文章。点击文章标题可以在新标签页中打开文章。

```bash
user@fake-cli:~$ ls
[2025-01-15] 如何使用 CLI 工具
[2025-01-10] JavaScript 异步编程
[2025-01-05] 我的第一个终端项目
```

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- marked.js (Markdown 解析)

## 使用技巧

1. 点击终端区域的任意位置可以聚焦输入框
2. 输入命令后按回车键执行
3. 输入不存在的命令会显示错误提示

## 未来计划

未来版本将添加更多功能，如命令历史记录、自动补全等。
