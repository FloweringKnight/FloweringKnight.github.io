# ERP资产管理系统

## 系统概述

企业资源计划(ERP)资产管理系统是一个完整的资产管理解决方案，实现了资产从采购、入库、领用、调拨、维修、盘点到报废的全生命周期管理。

## 功能模块

### 📊 首页工作台 (index.html)
**功能特点：**
- 资产统计概览：总资产、在用资产、闲置资产、维修中
- 资产价值趋势图表（柱状图展示12个月数据）
- 资产分类占比饼图（办公设备40%、生产设备24%、IT设备12%等）
- 最近操作记录（新增、更新、盘点、报废）
- 待办事项列表（审批、维修、盘点等任务）

**交互功能：**
- 统计卡片可点击，跳转到资产列表页查看详细数据
- "新增资产"按钮跳转到资产入库页面
- 响应式设计，支持移动端访问
- 登录状态检查（未登录自动跳转登录页）

### 📦 资产台账 (asset-list.html)
**功能特点：**
- 完整的资产列表展示（资产编号、名称、分类、部门、负责人等）
- 高级筛选面板（支持按分类、状态、部门、品牌、日期范围、价值范围筛选）
- 批量操作功能（导出、打印标签）
- 分页显示（显示1-5条，共2,847条记录）
- 表格排序和搜索功能

**交互功能：**
- 表格行可点击，跳转到资产详情页查看完整信息
- 复选框和操作按钮阻止事件冒泡，不影响行点击
- "新增资产"按钮跳转到资产入库页面
- 响应式设计，移动端支持横向滚动
- 登录状态检查

### 📋 资产详情 (asset-detail.html)
**功能特点：**
- 资产照片展示（主图和缩略图）
- 基本信息卡片（资产编号、名称、分类、品牌、规格等）
- 财务信息（资产原值、折旧、净值、残值）
- 资产状态时间线（入库、领用、盘点、维修记录）
- 操作历史记录
- 快捷操作按钮（调拨、借用、维修、报废）

**交互功能：**
- "返回列表"按钮回到资产台账页面
- "借用"按钮跳转到资产领用申请页面
- 其他快捷操作按钮弹出提示（待开发）
- 二维码展示区域（占位）
- 登录状态检查

### ➕ 资产入库 (asset-add.html)
**功能特点：**
- 完整的资产信息录入表单
- 供应商选择和采购单号关联
- 资产分类选择（办公设备、生产设备、IT设备等）
- 规格参数配置
- 附件上传功能
- 必填项验证（红色星号标识）

**交互功能：**

- "提交"按钮提示成功并返回列表页
- 表单验证和反馈（符合需求文档规范）
- 登录状态检查

### 📝 资产领用 (asset-borrow.html)
**功能特点：**
- 领用申请表单
- 资产选择（支持搜索和筛选）

- 使用部门选择

**交互功能：**

- 登录状态检查

### 🔍 资产盘点 (inventory-check.html)

**交互功能：**

## 页面间导航

系统实现了完整的页面间跳转功能：

1. **首页 → 资产列表**: 点击统计卡片或"新增资产"按钮
2. **资产列表 → 资产详情**: 点击表格任意行
3. **资产详情 → 资产列表**: 点击"返回列表"按钮
4. **资产列表 → 资产入库**: 点击"新增资产"按钮
5. **资产列表 → 资产领用**: 在详情页点击"借用"按钮
6. **各页面 → 资产盘点**: 通过侧边栏导航
